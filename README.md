# mini-react

A mini clone of React to learn about its inner workings at a high level.

## Features

- [x] `createElement`
- [x] Renderer (DOM)
- [x] Reconciler (Fiber)
- [x] Function components
- [ ] Hooks
  - [x] `useState`
  - [ ] `useEffect`
  - [ ] `useRef`
- [ ] Class components + lifecycle methods
- [ ] Key-based reconciliation
- [ ] Refs
